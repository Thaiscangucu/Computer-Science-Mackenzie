# Computer Science - Mackenzie

Welcome to my academic portfolio! This repository highlights key projects developed during my Computer Science degree at Mackenzie Presbyterian University, showcasing my progression across different languages, data structures, and algorithmic paradigms.

## Project Catalog

### Semester 1: Introduction to Programming
* **Mental Health Quiz:** An interactive terminal quiz focused on mental health education, utilizing control flows and data validation. *(Python)*

### Semester 3: Data Structures I
* **Mathematical Expression Evaluator:** Evaluates infix/postfix expressions using custom stack and queue implementations. *(Java)*
* **Assembly REPL Interpreter:** A custom execution environment simulating Assembly instructions using Linked Lists and Circular Queues. *(Java)*

### Semester 4: Data Structures II & Algorithms
* **Binary Expression Tree:** Parses arithmetic expressions into a binary tree to visualize traversals and recursively calculate results. *(Java)*
* **Sorting Algorithms Comparison:** Analyzes and compares the operational complexity of Insertion Sort versus Merge Sort. *(C)*

### Semester 5: Advanced Algorithms & Functional Programming
* **LZW Compression Algorithm:** Implements the lossless dictionary-based Lempel-Ziv-Welch compression method with dynamic memory management. *(C)*
* **Huffman Coding Compression:** A functional programming approach to text compression using a frequency-based binary prefix tree. *(Haskell)*

### Semester 6: Formal Languages & Compilers
* **Recursive Descent Parser:** Performs syntactic analysis and error-trapping on mathematical expressions based on a Context-Free Grammar. *(C)*
* **SAL Compiler Lexical Analyzer:** A custom state-machine token scanner and CLI for the Simple Algorithmic Language. *(C)*
