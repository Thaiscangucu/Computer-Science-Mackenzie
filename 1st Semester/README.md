# Mental Health Quiz

This project is an interactive terminal application developed in Python designed to test and expand knowledge about mental health. The quiz covers relevant topics such as global statistics, myths and facts about psychological disorders, and habits that influence emotional well-being.

## About the Project

The software serves as an educational tool. At the end of each question, if the user provides an incorrect answer, the program displays an explanatory note based on real data, promoting immediate learning.

### Key Features
* **Data Validation:** A system that ensures the user enters only valid commands (such as "INICIAR" or the options A, B, and C).
* **Corrective Feedback:** Detailed explanations provided after incorrect answers to clarify the topic.
* **Performance Calculation:** A final evaluation based on total correct answers, classifying the user's level of knowledge.

## Technologies Used

* **Language:** Python 3.x
* **Applied Concepts:** Repeat loops (while), conditional structures (if/elif/else), and input data handling.

## How to Run the Project

1. **Prerequisites:** You must have Python 3 installed on your machine.

2. **Execution:**
   Open the terminal in the folder where the file is located and run the following command:
   ```bash
   python "QUIZ-SAUDE MENTAL.py"
   ```
## How to play
* Type "INICIAR" to start the challenge.
* Use the letters A, B, or C to select your answers.
